git config --global user.name blabla додавання ім'я користувача
git config --global user.email blabal@bla додавання електронної пошти користувача
git init ініціалізація каталогу
git status поточний статус Git'a
git add вибір файлів (add . - вибір всіх файлів)
git commit комітація (-m "назва Коміту")
git log Shows the commit logs
git checkout Switch branches or restore working tree files
git branch List, create, or delete branches
git push завантаження на Github локальної версії файлів
git pull копіювання репозиторії із Github
git merge Join two or more development histories together
git clone Clone a repository into a new directory
.gitignore Specifies intentionally untracked files to ignore

ADVANCED LEVEL:
git revert Revert some existing commits
git reset Reset current HEAD to the specified state
git rebase Reapply commits on top of another base tip
git cherry-pick Apply the changes introduced by some existing commits
git stash Stash the changes in a dirty working directory away
